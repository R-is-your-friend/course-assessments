Access the Data

Important.Rdata, what’s this?

So far we have used data built into R (accessing using the data() function), or from .csv or .xlsx filter (accessed using readr functions). There is a special data type that is only used with R with the suffix .Rdata, which we will use in this lab.

The benefits of saving data that you are working with as .Rdata are:

  1. You can save as many named objects as you want in one file! In this case, there are 8 data frames saved in 08-bCH_murder_data.Rdata
  2. .Rdata files are much smaller (i.e. take up less space on your computer) than the same data saved as a .csv.

You “load” .Rdata files into your Global Environment in R using the load() function. All objects included in the .Rdata file will be loaded with the name they were saved with.

Save the 08-bCH_murder_data.Rdata file to your course directory and edit the file path in the code chunk below as needed to load in the data.

library(tidyverse)

load("08-bCH_murder_data.Rdata")
library(tidyverse)

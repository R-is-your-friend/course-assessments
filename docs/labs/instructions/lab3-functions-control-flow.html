Function 4: shorten()

13. Write code to manipulate the vector of numbers nums. Your code should use a while() loop to find the first cumulative sum GREATER THAN 10.

Tip

The \(i^{th}\) cumulative sum of a vector is the sum of element \(1\) through element \(i\).

Consider the vector \((2,4,6,8,10)\). The first cumulative sum is \(2\); the second cumulative sum is \(2 + 4 = 6\); the third cumulative sum is \(2 + 4 + 6 = 12\), etc.

14. Write a function called shorten(). This function should take in a vector, drop all values BEFORE the cumulative sum is greater than a provided number, and return the remaining values from the original vector. Adapt your while() loop from Q13 to do this.

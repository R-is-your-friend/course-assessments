Problem A - Fizz Buzz

Suppose you have a vector of numbers (e.g., 1:100). For each value, you want to print the following output:

  • if a number is divisible by 3, print "Fizz",
  • if the number is divisible by 5, print "Buzz",
  • if the number is divisible by both 3 and 5, print "FizzBuzz",
  • if none of these are true then print the number.

Pencil it Out

Brainstorm how you would write R code to solve this problem. Your solution must use a for()-loop.

num <- 1:100

for(i in 1:length(num)){
  # Numbers divisible by both 3 and 5
  if(num[i] %% 3 == 0 & num[i] %% 5 == 0){
    print("FizzBuzz")
    } 
  # Numbers divisible by 5
  else if(num[i] %% 5 == 0){
    print("Buzz")
    }
  # Numbers divisible by 3
  else if(num[i] %% 3 == 0){
    print("Fizz")
    } 
  # Numbers not divisible by 3 or 5
  else{
    print(num[i])
    }
  }

Dog Sitter

total_wetfood <- 0
total_dryfood <- 0

for(i in 1:nrow(dogs)){
  # Small dogs
  if(dogs$size[i] == "small"){
    # Old dogs
    if(dogs$age[i] > 12){
      total_wetfood <- total_wetfood + 8
    } 
    # Younger dogs
    else{
      total_dryfood <- total_dryfood + 8
    }
  } 
  # Medium dogs
  else if(dogs$size[i] == "medium"){
    # Older dogs
    if(dogs$age[i] > 12){
      total_wetfood <- total_wetfood + 16
    } 
    # Younger dogs
    else{
      total_dryfood <- total_dryfood + 16
      }
  } 
  # Large dogs
  else{
    # Old dogs
    if(dogs$age[i] > 12){
      total_wetfood <- total_wetfood + 32
    } 
    # Younger dogs
    else{
      total_dryfood <- total_dryfood + 32
      }
    }
}

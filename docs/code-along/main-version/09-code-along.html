Goal: Calculate the percent of the time of the grant that was taken away by it being terminated.

  2. Calculate the number of days that should have remained of the grant after the termination letter was sent.

  3. Use these two variables to calculate the percent of days taken from the grant recipients.

# Q1-3 code
nsf_terminations <- nsf_terminations |> 
  mutate(intended_length = as.period(nsf_expected_end_date - 
                                       nsf_startdate),
         missing_length = as.period(nsf_expected_end_date -
                                      termination_letter_date),
         lost_perc = missing_length / intended_length)
  1. What was the average and total amount of lost time by state in terms of years?
nsf_terminations |> 
  group_by(org_state) |> 
  summarize(mean_years = mean(missing_length / years(1)),
            total_years = sum(missing_length / years(1)),
            n = n()) |> 
  arrange(desc(total_years))

Strings

One of the variables in our data is in_cruz_list, which is a reference to a report by Ted Cruz, a US senator from Texas, which labeled 3,483 NSF grants as “questionable projects that promoted diversity, equity, and inclusion (DEI) tenets or pushed onto science neo-Marxist perspectives about enduring class struggle.” There is also a rebuttal report.

The report lists a large number of key words that the Cruz report used to identify these grants. Let’s see if we can find these terms in the cancelled grants. We compiled the terms into an excel sheet to make things easier.

search_terms <- read_xlsx("../../data/09-cruz_terms.xlsx")
  1. In order to figure out if any one of these terms is found in the grant title or abstract, we need to condense the term column in search_terms into a single string, call it, term_string where…
  1. each term is all lowercase
  2. each term is separated by a pipe “|”
term_string <- search_terms |> 
  mutate(term = str_to_lower(term)) |> 
  pull() |> 
  str_flatten(collapse = "|")

## Strings

One of the variables in our data is `in_cruz_list`, which is a reference to a [report](https://democrats-science.house.gov/imo/media/doc/democratic_staff_report-defending_hidden_figures.pdf) by Ted Cruz, a US senator from Texas, which labeled 3,483 NSF grants as  "questionable projects that promoted diversity, equity, and inclusion (DEI) tenets or pushed onto science neo-Marxist perspectives about enduring class struggle." There is also a [rebuttal report](https://www.govinfo.gov/content/pkg/GOVPUB-Y4_C73_7-PURL-gpo234941/pdf/GOVPUB-Y4_C73_7-PURL-gpo234941.pdf). 

The report lists a large number of key words that the Cruz report used to identify these grants. Let's see if we can find these terms in the cancelled grants. We compiled the terms into an excel sheet to make things easier.

```{r}
search_terms <- read_xlsx("../../data/09-cruz_terms.xlsx")
```

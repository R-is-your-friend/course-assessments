Everyone is obsessed with protein and fiber these days!

Goal: A table of the number of cereals that are low/medium/or high protein and low or high fiber.

  1. Create a variable called protein_level labeling each cereal as “low”, “medium”, or “high” protein, using the following categorizations:
  • low: 0-1 g protein
  • medium: 2-4 g protein
  • high: more than 4 g protein
  1. Create a variable called fiber_level labeling each cereal as “low” or “high” fiber, using the following categorizations:
  • low: 0-2 g fiber
  • high: more than 2 g fiber

  1. Find the number of cereals for each combination of protein and fiber level.

  2. Is this table easy to read? What might help?

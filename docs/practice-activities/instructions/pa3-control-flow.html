Practice Activity 3: Control Flow

Practice with Loops & Conditionals

Author

INSTRUCTIONS

The baRistas at Tidy Roasters have received a huge drink order and are preparing to fill it. Your job is to help summarize, and fix, the orders.

Part Zero: Accessing the Practice Activity

This week there are two files you need to download:

  1. Download the template Practice Activity R script here: pa-3-control-flow.R

  2. Download the coffee order data here: coffee_orders.xlsx

Important

Be sure to save both files inside your Week 3 folder of your STAT 1810 folder!

Part One: Getting the Drink Orders

  1. Let’s get the drink order data into R. Since we haven’t covered reading in data yet, we’ll make it easy! Follow these steps.

    1. In RStudio, go to the Files pane and navigate to the saved data.

    2. Click on the file name and select Import Dataset….

    3. Keep all of the settings and click Import.

You should now see the coffee_orders dataset in your Environment tab (upper right corner).

Part Two: Getting Information About the Drink Orders

  1. How many drinks are in the order? Save this number in an object named num_orders and print out the value.
Tip

Hint: Think about what dimension of the data correspond to the number of drinks. What R function measures that dimension?

  1. Think back to last week. What operator allows you to grab a single column from a dataframe? Using that operator, write a line of code that produces a vector of the drink Base for all drinks.

  2. Now write a line of code that produces the drink Base for the fourth drink order.

Part Three: Finding Specific Drink Orders

  1. Use a while() loop to find the first person to order a drink with a chai Base. Report this person’s full drink order.
Tip

Hint: Look over the handout to identify components you need for a while() loop!

  1. Use a for() loop to count the total number of espresso drinks.
Tip

Hint: You will need to use a conditional statement inside your for() loop!

Part Four: Making Drink Modifications

  1. Oh no! Tidy Roasters has run out of soy milk. The baRistas decide to use oat milk instead. In the coffee_orders dataframe, change all the soy milk requests to oat milk. Use the if_else() function from dplyr to do this.
Tip

Hint 1: Remember, you can find the documentation for a specific function by running ?FUNCTION_NAME in the console.

Hint 2: Remember to save the updated Milk back into the coffee_orders dataframe! This is the same step you used in last week’s PA!

  1. The baRistas need to know how much ice to have on hand. For a medium iced drink, they need 4oz of ice. For a large iced drink, they need 6oz of ice. Use a for() loop and conditionals to calculate the total ice amount the baRistas need.
